Weighted Least Squares Analysis

Julia Statistical Analysis

Project Overview

Implementation of weighted least squares regression with comprehensive visualization of results using Julia. This project demonstrates advanced statistical modeling techniques and their applications in data analysis.

Mathematical Foundation

Linear Algebra Statistical Theory Optimization

The weighted least squares method extends ordinary least squares by incorporating weights to handle heteroscedasticity:


# Julia implementation of weighted least squares
function weighted_least_squares(X, y, w)
    # X: design matrix
    # y: response vector
    # w: weights vector
    W = Diagonal(w)
    β = inv(X' * W * X) * X' * W * y
    return β
end

Implementation Details

Julia Plots.jl LinearAlgebra.jl

Key components of the implementation include:

  • Data preprocessing and weight calculation
  • Model fitting and parameter estimation
  • Residual analysis and diagnostics
  • Interactive visualization of results

using Plots
using LinearAlgebra
using Statistics

# Generate synthetic data with heteroscedastic errors
n = 100
x = range(0, 10, length=n)
y_true = 2 .* x .+ 1
ϵ = randn(n) .* (0.5 .+ x/5)
y = y_true + ϵ

# Calculate weights (inverse of variance)
weights = 1 ./ (0.5 .+ x/5).^2

# Fit weighted least squares model
X = hcat(ones(n), x)
β_wls = weighted_least_squares(X, y, weights)

Visualization

Plots.jl StatsPlots.jl

The project includes several types of visualizations:

  • Scatter plots with fitted lines
  • Residual diagnostic plots
  • Weight distribution visualization
  • Comparison with ordinary least squares

# Create visualization of results
plot(x, y, seriestype=:scatter, 
     label="Data", alpha=0.5,
     markersize=weights*50)
plot!(x, X * β_wls, 
      label="WLS fit",
      linewidth=2)
xlabel!("x")
ylabel!("y")
title!("Weighted Least Squares Regression")

Applications

The weighted least squares method was applied to several real-world datasets:

  • Economic data with varying reliability
  • Scientific measurements with known error margins
  • Time series data with changing variance
  • Cross-sectional studies with heteroscedastic errors

Key Findings

  • WLS provides more accurate estimates than OLS for heteroscedastic data
  • Proper weight selection is crucial for model performance
  • Visual diagnostics help in identifying model adequacy
  • Julia's performance enables efficient analysis of large datasets
