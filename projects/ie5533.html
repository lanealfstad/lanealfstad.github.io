Regression Analysis

R tidyverse caret

Implementation of various regression techniques:

  • Linear regression with regularization
  • Polynomial regression and splines
  • Cross-validation and model selection

# Example from Assignment 2
library(tidyverse)
library(caret)

# Fit polynomial regression
poly_fit <- train(
  quality ~ poly(density, degree = 3),
  data = wine_data,
  method = "lm",
  trControl = trainControl(method = "cv", number = 5)
)
